Dicionário HTML

Head

Essa tag é um contêiner para metadados, colocada entre a tag html e a tag body.


Body

Essa tag contém todos os conteúdos de um documento HTML.


Style

Essa tag é usada para definir estilos (CSS) para um documento HTML.


Script

Essa tag é usada para incorporar um script do lado do cliente (JavaScript).


Header

Essa tag é usada definir um cabeçalho para uma página.


Div

Essa tag é usada como contêiner para elementos de HTML.


Títulos

Essa tag adiciona títulos atribuindo ordem de importância de h1 até h6.


Parágrafos

Essa tag define um parágrafo.


Listas Ordenadas

Essa tag define uma lista ordenada, podendo ser numérica ou alfabética.


Listas Desordenadas

Essa tag define uma lista desordenada (com marcadores).


Imagens

Essa tag é usada para incluir uma imagem à uma página HTML.


Vídeos

Essa tag é usada para incorporar conteúdo de vídeo em um documento.


Links

Essa tag cria um hyperlink para uma outra página.


Botões

Essa tag cria um botão clicável.


Tabelas

Essa tag define uma tabela de HTML.
